Welcome to the **GENAI_drughunter** repository! This is a comprehensive series of 10 agentic and deep learning projects focused on revolutionizing the drug discovery pipeline using Generative AI, Multi-Agent Systems (LangGraph), and Retrieval-Augmented Generation (RAG).

## üìÅ Projects Overview

1.  **[MoleculeGPT](./project_1_MoleculeGPT)**: De Novo Drug Design via Fine-Tuned LLM + RLHF.
2.  **[ToxiSense](./project_2_ToxiSense)**: Multi-Modal Toxicity Prediction Agent (GNNs + ChemBERTa-2 + RAG).
3.  **[TargetRAG](./project_3_TargetRAG)**: Target Retrieval-Augmented Generation Agent for identification & validation.
4.  **[ADMET-Oracle](./project_4_ADMET_Oracle)**: Agentic ADMET Profiling System using stateful multi-agent workflows.
5.  **[ClinTrialGPT](./project_5_ClinTrialGPT)**: Clinical Trial Protocol Generator + Failure Risk Predictor.
6.  **[ProteinBind-LLM](./project_6_ProteinBind_LLM)**: Protein-Ligand Binding Affinity Predictor.
7.  **[LitMine-AI](./project_7_LitMine_AI)**: Autonomous Drug Literature Intelligence System.
8.  **[SideEffect-Navigator](./project_8_SideEffect_Navigator)**: Post-Market Drug Safety Signal Detection.
9.  **[RepurposeAI](./project_9_RepurposeAI)**: Drug Repurposing Engine using Multi-Hop Graph Reasoning.
10. **[DrugHunter-OS](./project_10_DrugHunter_OS)**: Full End-to-End Drug Discovery Operating System.
